# Task-2-Data-Visualization
Internship Task 2:Superstore Sales &amp; Profit Dashboard using Power BI.

Dataset Used:Superstore Sales Dataset

Tool Used:Power BI

Objective:To create meaningful visualizations that convey a compelling business story using sales data.

Dashboard Insights:
1. West region generates the highest sales and profit.
2. Technology category contributes the most to profitability.
3. Furniture category shows comparatively low profit.
4. Segment filter enables analysis across different customer groups.

Visualizations Included:
1. Sales by Region
2. Profit by Category
3. Profit by Region
4. Customer Segment Filter

Outcome:An interactive dashboard was created to highlight key sales and profit trends, helping identify high-performing regions and categories for better business decision-making.
